Barbero Inmobiliaria

Creación de sitio completo para Barbero Inmobiliaria

Tecnologías

WordPress MySQL PHP

Vista Previa

Barbero Inmobiliaria es una Inmobiliaria ubicada en Bahia Blanca, el sitio web le pemite al administrador crear, actualizar y eliminar propiedades facilmente desde un panel de control fácil e intuitivo gracias a WordPress. Las propiedades tienen categorias que luego son utilizadas para organizar las propiedades en el frontend, cada propiedad puede también tener una galeria de imágenes y un mapa personalizado de Google.

Sistema de Gestor de Contenido

CMS personalizado creado con .NET y Web Services

Tecnologías

C# .NET Javascript

Vista Previa (private system)

Un gestor de contenido personalizado creado en Paradigma, esta compuesto de un backend en ASP.NET con un ORM (TinyORM) personalizado, incluye creación de contenido, galerias de imágenes, gestion de archivos y newsletter. El backend expone un API en JSON en forma de Web Service, luego el frontend lo consume para generar una página personalizada a cada cliente.

Fisa Backend

Panel de administración para FISA

Tecnologías

CodeIgniter MySQL PHP Javascript

Vista Previa (private system)

Fisa es la expo más grande del sur de Argentina, cumple el proposito de fomentar la vinculación entre los distintos actores de la producción, la industria, el comercio, centros de estudios, trabajadores, organismos gubernamentales y no gubernamentales. El backend fue diseñado para permitir a los administradores gestionar todos los stands de la expo, y proveer al dueño de cada stand una interfaz web mediante la cual puede gestionar su stand, reduciendo la burocracia del sistema en general.

Proyectos Personales

Blueberry

Blueberry (arándano) es un lenguaje de programación basado en la sintaxis de Ruby, que compila a PHP, fue creado utilizando Node y Javascript. Es mayormente para uso experimental ya que actualmente no es un lenguaje estable.

Ultra DB

Ultra DB es una base de datos flat-files en PHP, esta diseñada para ser rápida, facil de usar e instalar. Es segura, y utiliza agresivamente el cache para alcanzar una performance óptima.

Katar

Katar es un motor de templates para PHP 5.x simplista y no agresivo, está basado en Blade. Katar no se mete en tu camino, puede ser mezclado con PHP, es rápido, fácil de configurar y usar en cualquier proyecto.

StudioWP Google Maps

StudioWP Google Maps es un plugin de WordPress que permite al usuario agregar un mapa de google fácilmente en cualquier parte de la entrada, solo especificando la dirección. No necesita ninguna configuración extra.

Alerter

Alerter es una implementación simple de alertas no agresiva, al ser creada con "vanilla javascript" no tiene dependencias y es muy facil de usar y extender.

DateTime Picker

Un plugin de jQUery UI que permite al usuario final seleccionar una hora a la vez de una fecha, es fácil de incluir a cualquier proyecto existente. Fue creado ya que las soluciones alternativas eran o no muy buenas o demasiado complicadas.

TinyORM

TinyORM es un ORM simplista para C#, funciona con cualquier base de datos que utilize IDbConnection, es solo un archivo, y muy facil de integrar a cualquier proyecto.

Sobre mí

  • Senior .NET programmer, Senior web developer

Desarrollador web de Argentina con un amor especial por el open source, las buenas prácticas, los diseños simples y hermosos y los gatos.

He estado jugando con las tecnologías web desde que tenia 15 años, en ese entonces PHP estaba en auge y nadie sabía como crear proyectos mantenibles y ágiles.

Crear sitios simples, hermosos y funcionales es lo que amo, y por esa razón, siempre estoy interesado en los ultimos trends del diseño y UX.

A medida que la web evolucionaba, yo también lo hice, cuando el movimiento MVC web comenzó con Rails, experimente varios frameworks en PHP como CodeIgniter y Symfony, también aprendí varias tecnologías como el framework .Net, Rails y Node.

A medida que Javascritp ganaba popularidad decidi investigar en profundidad la tecnología, encontrando un lenguaje simple, muy potente y dinámico realmente capturó mi interés, en especial la parte funcional del lenguaje. Tengo experiencia con jQuery, jQuery UI, y frameworks MV* como Angular.

En el 2012 deje de hacer freelance y entre full-time en Paradigma, donde estaba a cargo del departamento de desarrollo web. En paradigma trabajé con .NET, incluyendo ASP.NET MVC y Clásico, y SQL Server.

Actualmente estoy estudiando y abierto a trabajo freelance, ofreciendo el paquete completo que la web moderna necesita, diseños simples, funcionales y responsivos, programación y hosting.
